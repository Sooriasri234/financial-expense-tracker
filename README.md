
# Financial Expense Tracking & Analytics

An intelligent Financial Expense Tracking and Analytics System built using Streamlit, PostgreSQL, Machine Learning, and Python. The application helps users manage expenses, monitor budgets, analyze spending patterns, and predict future financial expenses using AI/ML techniques.

## 📌 Features

### 🔐 Authentication System
- User Registration
- Secure Login & Logout
- JWT-based Authentication

### 💰 Expense Management
- Add Daily Expenses
- Categorize Transactions
- Income & Expense Tracking
- Transaction History

### 📊 Financial Analytics Dashboard
- Interactive Charts
- Monthly Expense Analysis
- Category-wise Spending Visualization
- Financial Summary Reports

### 🚨 Budget Alert System
- Set Monthly Budgets
- Budget Warning Alerts
- Budget Exceeded Notifications
- SMTP Email Notifications

### 🤖 AI & Machine Learning
- Expense Prediction using Linear Regression
- Future Expense Forecasting
- Spending Pattern Analysis
- Smart Financial Insights

### 📈 Financial Insights
- Monthly Expense Comparison
- Expense Increase/Decrease Detection
- Savings Analysis
- Personalized Financial Recommendations
