
# Financial Expense Tracker

A Machine Learning based Financial Expense Tracking and Analytics System built using Streamlit, Python, MySQL/SQLite, and Scikit-learn.

---

## Features

- User Authentication (Login/Register)
- Expense Tracking
- Monthly Budget Management
- SMTP Email Budget Alerts
- AI Expense Prediction
- Smart Financial Insights
- Monthly Expense Comparison
- Interactive Charts and Analytics Dashboard

---

## Tech Stack

- Python
- Streamlit
- MySQL / SQLite
- Pandas
- Plotly
- Scikit-learn
- SMTP
- JWT Authentication

---

## Machine Learning

The project uses Linear Regression to predict future expenses based on historical financial transaction data.
